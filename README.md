# i3 DotFile
---

My i3 WM config files. including `compton`, `i3` and `i3blocks`.   

## Usage
```bash
git clone https://github.com/stkevintan/i3dotfile  ~/.config/i3
```
Dependencies(Part) :
```yaml
wifi:
  - nmcli
temperature:
  - sensors
i3:
  - i3blocks
  - compton
  - dmenu
  - i3-dmenu-desktop
  - quickswitch.py
  - rofi
  - xfce4-appfinder
  - nitrogen
```
